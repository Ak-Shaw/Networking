Question 1: What does IPV4 stand for ?
Answer: Internet Protocol Version 4

Question 2: In which format are IP addresses represented?
Answer : IP addresses are represented in Dotted- Decimal format


Question 3: IP addresses consist of how many bits.What is the equivalent number of bytes ?
Answer: IP addresses consist of 32 bits. That is, 4 bytes.

Question 4:  Consider the IP address 193.45.23.2 . What is the binary representation of this address?
Answer: 11000001 00101101 000101111 00000010

Question 5: Consider this binary representation : 10100100 00110100 11011010 00101010. What is the decimal representation of this address?
